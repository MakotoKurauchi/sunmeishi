# sun meishi

![sun meishi](https://i.imgur.com/4IJk04T.jpg)

## これは

Kailh Sun Switchを使用した名刺サイズのマクロパッドです。
ロータリーエンコーダもLEDが入っているのでカラフルに光ります。

## 課題

現状でも動作には問題ありませんが、LEDドライバをISSIのデバイスに変えたいと思っています。
